# QA Portfolio: Conduit Project

## Overview
This repository contains testing artifacts and documentation for the Conduit application. It serves as a portfolio piece to demonstrate proficiency in the software testing lifecycle, including test suite decomposition, manual test execution, and defect tracking.

## Tools & Tech Stack
* **Project Management:** Jira
* **Test Management:** TestRail

## Testing Strategy
### 1. Test Suite Decomposition
The project was structured into two main functional areas to ensure comprehensive coverage:
* **Disconnected User**
    * Home page, Sign In, Sign Up, Article, and User pages.
* **Connected User**
    * Home page, New Article, Settings, User, and Article pages.

![Test Suite Decomposition](Test%20Suite%20Decomposition.png)

### 2. Test Execution
* **Methodology:** Manual testing execution following strict functional requirements.
* **Results:** 100% pass rate achieved for identified critical test paths.
